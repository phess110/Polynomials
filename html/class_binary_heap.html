Detailed Description

template<typename K, typename V, typename Compare>
class BinaryHeap< K, V, Compare >

Templated class for a binary heap data structure.

Template Parameters
Kthe key data type
Vthe value data type
CompareA struct containing a callable comparision operator

The heap is implemented using a vector of key-value pairs
