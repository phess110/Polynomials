Detailed Description

Represents a polynomial as a list of coefficients.

Remarks
Proper Usage: While input coefficients are allowed to be doubles, it's best to only use this with integer coefficients.
For a polynomial with rational coefficients: \( f(x) = p_0/q_0 + ... + p_n/q_n x^n \), find the common denominator: \( d = \mathrm{lcm}(q_0,...,q_n) \). Factor out \( 1/d \) to write f as a polynomial with integer coefficients: \( f(x) = 1/d * (a_0 + ... + a_nx^n) \) Perform desired operations on \( a_0 + ... + a_nx^n \) and then scale output by \( 1/d \).

Computes the Lagrange Coefficients.

Parameters
[in]pointsA vector of point-value pairs \( \{(x_i,y_i)\} \)
Returns
A vector \( L_i(x) = y_i \prod_{j \neq i} \frac{1}{x_i-x_j} \)

Define \( P_i(x) = \prod_{j \neq i} \frac{x-x_j}{x_i-x_j} \). Then the interpolated polynomial is \( P_{interp} = \sum_{i=1}^{n+1} y_iP_i(x) \)

Note
For numerical stability, I don't optimize the calculation of the L_i's
